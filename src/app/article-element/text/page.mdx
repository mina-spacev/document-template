# 텍스트

텍스트 요소들을 사용하여 콘텐츠를 작성할 수 있습니다.

### 헤딩 (제목)

각 문단의 제목, 부제목을 작성할 때 사용합니다.

```tsx copy
/* 색상 강조 제목 */
<h2 class="highlight">{title}</h2>
```

```tsx copy
/* 일반 제목 */
<h2>{title}</h2>
```

```tsx copy
/* 색상 강조 부제목 */
<h3 class="highlight">{title}</h3>
```

```tsx copy
/* 일반 부제목 */
<h3>{title}</h3>
```

### 문단 (본문)

일반적인 텍스트 내용을 작성할 때 사용합니다.

```tsx copy
/* 일반 텍스트 */
<p>{title}</p>
```

```tsx copy
/* 캡션 텍스트 */
<p class="caption">{title}</p>
```

### 강조 텍스트

중요한 내용을 강조할 때 사용합니다.

```tsx copy
/* 강조 */
<strong>{text}</strong>
```

```tsx copy
/* 텍스트 색상 강조 */
<strong class="highlight">{text}</strong>
```

```tsx copy
/* 배경 색상 강조 */
<strong class="bg-highlight">{text}</strong>
```
