# 유튜브

YouTube 동영상을 콘텐츠에 포함할 때 사용합니다.

- `src` : YouTube 영상 주소를 작성합니다.
- `title` : 해당 영상을 설명할 수 있는 문구를 작성합니다. (ex. 제목)

```tsx copy
/* 영상 삽입 */
<div class="video">
  <iframe
    src={youtube}
    title={title}
    allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share"
    referrerpolicy="strict-origin-when-cross-origin"
    allowfullscreen
  ></iframe>
</div>
```

```tsx copy
/* 영상 삽입 및 하단 설명 */
<div class="video">
  <iframe
    src={youtube}
    title={title}
    allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share"
    referrerpolicy="strict-origin-when-cross-origin"
    allowfullscreen
  ></iframe>
  <p>
    {caption}
  </p>
</div>
```

### YouTube 영상 주소 가져오기

1. YouTube 영상에서 '공유' 버튼 클릭
2. '퍼가기' 버튼 클릭
3. src 영역 중 `?` 전까지 복사 (https://www.youtube.com/embed/XXXXX)
