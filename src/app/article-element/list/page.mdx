# 리스트

항목을 나열할 때 사용합니다.

### 순서 없는 목록 (Unordered List)

순서가 중요하지 않은 항목들을 나열할 때 사용합니다.

```tsx copy
<ul>
  <li>{text}</li>
  <li>{text}</li>
</ul>
```

### 순서 있는 목록 (Ordered List)

단계별로 진행되는 내용이나 순서가 중요한 항목들을 나열할 때 사용합니다.

```tsx copy
/* 순서 있는 목록 */
<ol>
  <li>{text}</li>
  <li>{text}</li>
  <li>{text}</li>
</ol>
```
