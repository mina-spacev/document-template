### 사용 방법

팔레트에 있는 색으로 배경색이나 텍스트 색상을 변경할 때에는 다음과 같은 방식을 사용합니다.

```tsx
/* 배경 색상 */
<div class="box bg-red-50">박스</div>;

/* 텍스트 색상 */
<p class="text-red-500">텍스트</p>;
```
<br/>

<Container>
  <Article>
    <Box className="bg-red-50 w-full">박스</Box>
  </Article>
  <p className="text-red-500">텍스트</p>
</Container>

그 외의 색을 사용하고 싶다면 직접 색상 컬러 코드를 지정합니다.

```tsx
/* 배경 색상 */
<div class="box bg-[#d2e2fc]">박스</div>;

/* 텍스트 색상 */
<p class="text-[#004420]">텍스트</p>;
```
<br/>

<Container>
  <Article>
    <Box className="bg-[#d2e2fc] w-full">박스</Box>
  </Article>
  <p className="text-[#004420]">텍스트</p>
</Container>

위의 방법이 동작하지 않을 경우 아래와 같은 방식을 사용합니다.

```tsx
/* 배경 색상 */
<div class="box" style="background-color: #d2e2fc;">박스</div>;

/* 텍스트 색상 */
<p style="color: var(--color-green-900);">텍스트</p>;
```
<br/>

<Container>
  <Article>
    <Box class="box w-full" style={{ backgroundColor: "#d2e2fc" }}>박스</Box>
  </Article>
  <p style={{ color: "var(--color-green-900)" }}>텍스트</p>
</Container>

### 사용 팁

- 배경색은 일반적으로 `bg-[color]-50` 컬러를 사용하면 적절합니다.
- 하이라이트 색상은 일반적으로 `bg-[color]-500`, `text-[color]-500`을 많이 사용합니다.
