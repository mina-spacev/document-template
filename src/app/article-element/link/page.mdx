# 링크

다른 페이지나 외부 사이트로 연결하는 링크를 만들 수 있습니다.

- `href`: 이동할 URL 주소를 작성합니다.
- `target="_blank"`: 새 탭에서 창을 열도록 설정합니다.

```tsx copy
/* 색상 강조 + 밑줄 */
<a href={link} target="_blank" class="highlight underline">
  {text}
</a>
```

```tsx copy
/* 밑줄 */
<a href={link} target="_blank" class="underline">
  {text}
</a>
```

```tsx copy
/* 색상 강조 + 마우스 올렸을 때 밑줄 */
<a href={link} target="_blank" class="highlight hover:underline">
  {text}
</a>
```

```tsx copy
/* 마우스 올렸을 때 밑줄 */
<a href={link} target="_blank" class="highlight hover:underline">
  {text}
</a>
```

```tsx copy
/* 일부 내용 색상 강조 */
<a href={link} target="blank">
  {text} 
  <strong class="highlight">{text}</strong>
</a>
```
