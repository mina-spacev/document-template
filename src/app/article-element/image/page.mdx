# 이미지

이미지를 사용하여 시각적 콘텐츠를 추가할 수 있습니다.

- `src` : 이미지의 링크를 작성합니다.
- `alt` : 이미지를 설명할 수 있는 단어 또는 문장을 작성합니다.

### 단일 이미지

하나의 이미지를 표시합니다.

```tsx copy
/* 썸네일 이미지 */
<img src={image} alt={description} class="thumbnail" />
```

```tsx copy
/* 일반 이미지 */
<img src={image} alt={description} />
```

### 이미지 리스트

두 장의 이미지를 표시합니다.
PC 환경에서는 가로로, 모바일 환경에서는 세로로 배열됩니다.

```tsx copy
<div class="image-list">
  <img src={image} alt={text} />
  <img src={image} alt={text} />
</div>
```

### 이미지 (캡션 포함)

이미지와 함께 이미지에 대한 설명을 화면에 표시합니다.

```tsx copy
<figure>
  <img src={image} alt={text} />
  <figcaption>{caption}</figcaption>
</figure>
```

### 사용 팁

- **alt 속성**: 이미지에 대한 적절한 설명을 작성해주세요.
- **캡션**: 이미지의 내용을 설명하거나 출처를 작성할 필요가 있을 경우 사용합니다.
