# 템플릿 사용 가이드

아티클 및 이벤트 템플릿 사용법을 안내하는 Nextra 기반 문서 프로젝트입니다.

### 개발 서버 실행

```bash
pnpm dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 결과를 확인할 수 있습니다.

### 빌드 및 배포

```bash
# 프로덕션 빌드
pnpm build

# 프로덕션 서버 실행
pnpm start
```

### 템플릿 유형
- **아티클 템플릿**: 콘텐츠 아티클 작성을 위한 템플릿
- **이벤트 템플릿**: 이벤트 페이지 작성을 위한 템플릿

### 주요 기능
- 템플릿 요소 및 작성 가이드, 예시 코드 제공
- 코드와 렌더링 결과를 동시에 확인 가능

### 기술 스택

- **[Next.js 16](https://nextjs.org)** - React 프레임워크
- **[Nextra 4](https://nextra.site)** - 문서 사이트 생성기
- **[Tailwind CSS 4](https://tailwindcss.com)** - CSS 프레임워크
